# Historical site snapshots (2008–2009)

Current documentation lives in the handbook at **[https://alorbach.github.io/ultrastats/](https://alorbach.github.io/ultrastats/)** and in the [main repository](https://github.com/alorbach/ultrastats). For a short timeline (early releases, dormant period, 2026 revival), see [Project history](project-history.md).

The original **ultrastats.org** site and **wiki.ultrastats.org** are gone. Links below are preserved as archive references only, not active support channels.
