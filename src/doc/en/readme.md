# Readme (bundled static doc)

Welcome to **UltraStats**, a log parser for **Call of Duty** dedicated server log files, with a web front end for statistics and an admin area.

> **Up-to-date** requirements, install, and development setup: see the repository [README.md](../../README.md) and [INSTALL](../../INSTALL) at the project root.

## Authors

- Andre Lorbach ([alorbach](https://github.com/alorbach/ultrastats) on GitHub)

## Supported games (original scope)

- Call of Duty  
- Call of Duty: United Offense  
- Call of Duty 2  
- Call of Duty 4: Modern Warfare  
- Call of Duty: World at War
